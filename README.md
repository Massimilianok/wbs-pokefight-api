# WBS Coding School project: Pokemon fight API

## Project summary

API to serve data, from MongoDB database, to [`wbs-pokefight-app`](https://github.com/Massimilianok/wbs-pokefight-app) project.

## Main technologies

- JavaScript
- Node.js

## Database

- MongoDB

## Main packages
- express
- dotenv
- jsonwebtoken
- bcrypt
- mongoose

## Database hosting

- MongoDB Atlas

## Hosting
